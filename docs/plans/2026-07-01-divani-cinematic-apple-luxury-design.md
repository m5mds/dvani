# Divani Cinematic Apple Luxury Design

## Approved Direction

Make the Divani website feel Apple-like through restraint, pacing, alignment, and premium product-style presentation, while keeping Divani's dark identity and interior-design character.

This should not become a white tech landing page. The target is a dark cinematic portfolio with Apple-level discipline.

## Design Principles

- One clear idea per section.
- Fewer visible containers and less card noise.
- Project imagery behaves like the product reveal.
- Copy is short, natural, and confident.
- Motion is slow, physical, and useful.
- Navigation and CTAs stay quiet until needed.
- Client logos remain intentional proof, never background decoration.
- Mobile should feel curated, not stacked.

## Page Rhythm

### Home

1. Hero as a controlled first scene:
   - Dark project image.
   - Right-anchored Arabic copy.
   - Minimal label, one sentence, two actions.
   - More breathing room around the text.

2. Client proof strip:
   - Quieter and more premium.
   - Less like a banner, more like a trust rail.

3. Project reveal:
   - The strongest proof becomes the main event.
   - One large image, one title, one short explanation.
   - Project controls feel like product tabs/spec selectors.

4. Services:
   - Move from card-heavy blocks toward concise rows/specs.
   - Less explanation, more clarity.

5. Quote close:
   - Simple, direct, confident.
   - No over-selling.

### Inner Pages

- Keep the same page structure but reduce visual noise.
- Page heroes should feel like chapter openings.
- Use calmer heading scale and tighter copy.
- Avoid adding new dense sections unless they improve clarity.

## Visual System

- Keep dark black/brown base and warm gold accents.
- Increase negative space.
- Use hairline borders and soft translucent surfaces sparingly.
- Replace repeated cards with rows, split panels, and feature blocks where possible.
- Preserve clean inspected imagery only.

## Motion System

- Page entry: soft fade.
- Hero image: subtle settle only.
- Section reveal: small lift and blur, staggered lightly.
- Project switch: smoother crossfade/scale, no abrupt flashing.
- Logo strip: slow and almost ambient.
- Respect reduced motion.

## Success Criteria

- The page feels quieter, richer, and more controlled.
- The projects feel like the main product.
- Text no longer feels awkward or overdramatic.
- No half-logo/background misuse.
- No visual chaos on mobile.
- Browser audit remains clean: links, images, overflow, console, forms, drawer, project switching.
