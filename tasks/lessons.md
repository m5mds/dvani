# Lessons

## Avoid Generic Client Presentation Pages

- When the user asks for a strong client-facing visual result, do not settle for a polished but familiar landing-page layout.
- Lead with a distinct art direction before editing: define the visual metaphor, composition system, interaction feel, and proof hierarchy.
- Use project-specific assets as the dominant design language, not as small gallery decoration.
- Remove or transform generic SaaS/portfolio patterns such as predictable hero-card-service-gallery sequences unless the client explicitly wants that style.

## Reduce Content Pressure For First-Time Clients

- A client-facing interior design page should feel clear before it feels comprehensive.
- If the user says the page has too much information, cut repeated proof blocks, reduce jargon, and turn dense scope/process details into simpler decision paths.
- Keep the premium art direction, but let the visitor understand the next step quickly: what the company does, what projects prove it, and how to start.

## Beauty Needs More Than Clarity

- After simplifying a premium visual site, re-add beauty through composition, material texture, light, image treatment, and motion, not by restoring copy density.
- If the user says the site is "not pretty enough", audit the visible screenshots for flat surfaces, harsh empty spaces, weak section transitions, and underused assets.
- Prefer layered image moments, refined typographic contrast, and tactile details over generic dark/gold styling.
- Do not let the hero carry the whole page. Each major section needs its own visual anchor, framing rule, and image/material moment so full-page screenshots feel designed end to end.

## Audit Before Adding Polish

- When the user says "this isn't it" after visual enhancements, stop making incremental CSS changes and switch to a senior design audit.
- Diagnose the creative direction, hierarchy, emotional tone, and brand desirability before proposing implementation.
- A luxury/interior-design site must feel intentional, sensual, spatial, and editorial; dark gold styling alone is not a premium concept.

## Calm Beats Chaos

- If the user likes the visual-first direction but says it feels chaotic, reduce simultaneous visual ideas instead of adding more polish.
- Use one dominant image per section beat, not overlapping collages, background images, thumbnails, and captions all competing at once.
- Premium interior design pages need restraint: bigger images, fewer fragments, clearer rhythm, and calmer transitions.

## Elegance Requires Clean Source Imagery

- Do not use portfolio spreads with embedded client logos, watermarks, or text as atmospheric backgrounds. They read as accidental and cheap, especially in hero/contact/footer areas.
- Do not reuse the same project image across multiple major moments unless it is deliberately treated as the signature identity image.
- Luxury typography should feel controlled. Oversized titles can be dramatic, but if every section screams, the page becomes chaotic and loses elegance.

## Preserve Working Project Proof

- When the user says one part of the page was good, preserve and refine that part while fixing the weaker surrounding sections.
- Do not replace an interactive project showcase with static cards unless the user explicitly asks for a simpler gallery.
- For an interior design portfolio, project proof is the conversion spine: it should be prominent, browsable, and visually disciplined.

## Audit Every Structural Change

- When adding or deleting a section, immediately audit the whole page rhythm: navigation anchors, content order, image reuse, heading scale, mobile flow, and conversion path.
- Do not make isolated visual edits on a client-facing page without checking whether the change weakens the portfolio proof or creates misalignment elsewhere.
- Finish end to end: implementation, consistency audit, browser verification, screenshots, and documented results.

## Use Only Clean, Complete Project Imagery

- Never use an image crop with half a client logo, cut-off branding, baked-in PDF text, visible slide artifacts, or accidental partial marks.
- Before placing any image, inspect it at the target crop and reject it if it looks like a broken screenshot instead of deliberate project proof.
- Client logos belong in intentional logo treatments only; project/hero/background imagery must be clean, complete, and visually controlled.

## Match The Brand Voice Before Adding Motion

- If the user says the content needs more work, do not only adjust visual styling; audit whether the wording sounds like the company, the market, and the client decision context.
- Apple-like should mean restraint, clarity, smooth pacing, and system-level polish, not copying Apple branding or turning the site into a tech product page.
- For Divani, pair refined motion and typography with an interior-design voice: composed, exact, tactile, operationally credible, and Arabic-first.

## Hero Placement And Natural Arabic Copy

- Do not let a premium landing hero default to centered template placement unless the composition explicitly calls for it.
- For Arabic luxury/service pages, hero copy should usually feel anchored and intentional: right-aligned, away from the visual center, and balanced against the image.
- Avoid dramatic Arabic lines that sound translated or theatrical. Prefer natural, confident phrasing that a real client can understand quickly.
